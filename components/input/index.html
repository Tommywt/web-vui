# Attributes 属性

参数 说明 类型 可选值 默认值
value/v-model 绑定value值 String * *
type input的type类型 String password text
placeholder input的placeholder String * *
disabled input禁用 Boolean * false
error 错误提示 String * *
success 成功提示 Boolean * *
